# Traffic Violations Data Analysis

## Overview

This project is a data analysis mini-project based on the **Traffic Violations Dataset**.
The goal of this project is to perform data cleaning, preprocessing, visualization, and analysis using Python.

The project was completed using **Google Colab / Jupyter Notebook**.

---

## Dataset Information

The dataset contains traffic violation records with multiple attributes such as:

* Date of Stop
* Time of Stop
* State
* Vehicle Type
* Driver Information
* Location (Latitude, Longitude)
* Violation Type
* Vehicle Year
* Gender

---

## Project Tasks Performed

### 1. Data Loading

* Imported CSV dataset into pandas DataFrame

### 2. Data Inspection

* Displayed first 5 rows (`head()`)
* Displayed last 5 rows (`tail()`)
* Checked shape of dataset
* Checked column names
* Checked data types
* Displayed dataset information

### 3. Data Cleaning

* Checked missing values
* Removed duplicate values
* Filled missing numeric values using median
* Filled missing categorical values using "Unknown"

### 4. Statistical Analysis

* Generated descriptive statistics
* Used `loc` and `iloc`
* Performed `groupby()` operations
* Counted categorical values

### 5. Data Visualization

Created different charts:

* Histogram
* Pie Chart
* Bar Chart
* Line Chart
* Heatmap
* Box Plot
* Violin Plot
* Scatter Plot
* Density Plot
* Area Chart

### 6. Export

* Saved cleaned dataset into new CSV file
